# iExplorer Bypass ☠️

### Manual Loader
1. Clone this repo `git@github.com:Raafas/iExplorerLauncher.git`
2. Compile
3. Open terminal, go to `Build/Products/Debug/`
4. Run `DYLD_INSERT_LIBRARIES=libHook.dylib /Applications/iExplorer.app/Contents/MacOS/iExplorer &`
